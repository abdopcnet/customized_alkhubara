# App File Structure (customized)

## Key Paths

- `customized/customized/hooks.py`: App hooks (includes, events, fixtures config).
- `customized/customized/public/js/customer.js`: Client script loaded on Customer DocType.
- `customized/customized/customer.py`: Whitelisted backend methods used by the dashboard/actions.
- `customized/customized/fixtures/`: Exported fixtures (JSON) for custom records.
